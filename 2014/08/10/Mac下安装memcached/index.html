安装libevent(memcached的依赖库)

  • 下载libevent源码
  • 解压tar -xzf libevent-2.0.21-stable.tar.gz
  • cd libevent-2.0.21-stable
  • sudo ./configure --prefix=/usr
  • sudo make
  • sudo make install

安装memcached

  • 下载memcached源码
  • 解压memcached-1.4.20
  • cd memcached-1.4.20
  • sudo ./configure --prefix=/usr
  • sudo make
  • sudo make install
