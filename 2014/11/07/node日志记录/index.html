Java与Nodejs中控制台输出及日志记录对比

该博客框架解析Markdown的table不是太好, 以下采用列举方式进行对比

Java

  • 控制台: System.out.println('hello');
  • 日志记录(自带): java.util.logging.Logger
  • 日志记录(第三方): slf4j(定义接口), log4j(实现类)

Nodejs

  • 控制台: console.log('hello');
  • 日志记录(自带): util.log
  • 日志记录(第三方): log4js
