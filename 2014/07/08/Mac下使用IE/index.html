如何在Mac上使用IE？

  1. 点击modern.IE，选择要下载的版本，点击Grab them all with cURL获取下载的URL，例如
  2. $ cd ~/Downloads/VisualBox_IE，切换到保存文件的目录；
  3. 执行获取的URL
  4. $ chmod +x ./IE11.Win7.For.MacVirtualBox.part1.sfx
  5. $ ./chmod +x ./IE11.Win7.For.MacVirtualBox.part1.sfx
  6. 双击IE11 - Win7.ova，并配置内存
  7. 完成

另一种方式：Mac安装IE 方法二,使用到googlecode需要使用代理。
